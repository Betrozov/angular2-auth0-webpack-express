# angular2-webpack

##Quick start

```
# clone our repo
git clone git@github.com:Betrozov/angular2-webpack.git

# change directory to our repo
cd angular2-webpack

# install the repo with npm
npm install

# install TypeScript typings
tsd install

# start the server
npm start

# start the webpack
npm run dev

go to http://localhost:8080 in your browser
```
